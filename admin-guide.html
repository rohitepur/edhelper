2. Managing Classes

Viewing Classes

The Classes tab shows a table with all classes, their join codes, schedules, and breaks.

Editing a Class

1. Click the class name in the table
2. An editor opens at the bottom with fields for:
  • Class Name — display name shown everywhere
  • Join Code — students need this to register
  • Start/End Date — class duration
  • Schedule — click "+ Add Day" for each class meeting (day + start/end time)
  • Breaks — click "+ Add Break" for holidays (name + start/end date)
  • Test Plan — add tests you plan to give (PSSA, Keystone, STARR, Regents, MathCounts)
3. Click "Save" when done

Adding a New Class

1. Scroll to the bottom of the Classes page
2. Type the class name (e.g. "Geometry")
3. Click "Add Class" — the editor opens immediately

Quick Assign from Classes

Each class row has an "Assign Test" button. Clicking it opens a dedicated assign page with the class pre-selected.

4. Assigning Tests

Standard Tests (PSSA / Keystone)

1. Go to "Assign Test" tab
2. Select "Assign By" — Grade, Class, or Individual Student
3. Select Test Type — PSSA or Keystone
4. Set grade, number of questions, test name (optional)
5. Set due date, available-at time (optional), time limit (optional)
6. Click "Assign Test"

Custom Tests

1. Go to "Assign Test" tab → click "Custom Test" sub-tab
2. Paste your problems (with A/B/C/D options and ANSWERS section)
3. Check "Make Dynamic" if you want numbers to change each time
4. Click "Process Questions"
5. Review the parsed questions, remove any bad ones
6. Set test name, due date, time limit
7. Click "Save & Assign"

Scheduling Tests

Use the "Available At" field to schedule a test for a future time. Students will see the test grayed out with a countdown until it unlocks.

Individual Student Assignment

You can assign tests to individual students from:

  • The Assign Test page (select "Individual Student")
  • A student's profile page (Assign Test tab)

5. Creating Custom Tests

Supported Formats

The parser handles two PDF formats:

Kuta Software format:

1) 28.2 × 1.7 × 0.3
a) 2.842
b) 14.382
c) 28.042
d) 143.82

ANSWERS
B

PSSA Sampler format:

1 An expression is shown below...
A – 0.651
B – 0.093
C 0.093
D 0.651

(Answer key in summary table at bottom)

Dynamic Questions

When "Make Dynamic" is checked, the system detects math operations in each question and randomizes the numbers. Students get different numbers each time they take the test.

Tip: Questions that can't be dynamized (conceptual questions, graph questions) are automatically kept static or filtered out.

Questions That Get Filtered Out

The parser automatically removes:

  • Graph/diagram questions ("shown below")
  • Number line questions
  • Box plot / histogram questions
  • Questions with no text content
  • Kuta Software copyright junk

9. Approving Students

Registration Flow

1. Student goes to the site and clicks "Register"
2. They fill in: name, grade, class, class code, parent email/phone, password
3. The class code must match what you set in Classes
4. Their request appears in the Pending tab
5. Click "Approve" or "Reject"
6. Once approved, the student can log in
Tip: You can bulk approve/reject using the checkboxes and bulk action buttons.

Offline Mode

The site has a service worker that caches all files. If WiFi drops during a test, the student can continue. Scores save to localStorage and sync to Firebase when connection returns.

14. Troubleshooting

Common Issues

ProblemSolution
Student can't log inCheck they're approved in Pending tab. Check password is correct in Roster.
Student doesn't see assigned testCheck the test is assigned to their class/grade. Check the "Available At" date hasn't passed yet.
Scores not showingWait 10-15 seconds for sync. Check browser console for errors.
Custom test questions look wrongThe PDF paste may have formatting issues. Use the Remove button to delete bad questions before saving.
Admin page not loadingHard refresh (Ctrl+Shift+R). Check internet connection.
Student forgot passwordThey can click "Forgot password?" on login. You'll see a reset request in Help Requests.
"Unknown skill" in reportsOlder scores may not have skill data. New scores will show proper skill names.
